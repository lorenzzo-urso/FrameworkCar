o que está por trás?

Não é um chatbot comum. Cada resposta é raciocinada sobre um modelo aberto do Código Florestal.

🧠 Ontologia

As regras do Código Florestal viram um grafo que o computador entende (APP, Reserva Legal, biomas). É o conhecimento.

🔎 Rastreabilidade

Toda resposta cita a lei de onde veio (ex: Art. 4º). A regra vem do grafo — o LLM não inventa.

🛰️ Validação antecipada

Cruza a geometria do imóvel com dados abertos e descobre o déficit de APP antes do envio.

🤖 Agent Hub

O Compadre é 1 de N agentes sobre a mesma base. Governo e cooperativas plugam os seus.
